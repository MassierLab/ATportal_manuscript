# adiposetissue.org Manuscript
This repository contains a detailed description of the data collection and methods used to establish the adipose tissue knowledge portal [adiposetissue.org](adiposetissue.org). Additionally, scripts used to generate the plots in the manuscript are collected here. The code for the portal itself is also available and can be found here:  [AT portal github]https://github.com/jiawei-zhong/ATportal.
